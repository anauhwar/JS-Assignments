Methods of String and Array

Methods of string:
1) search(): Searches a string for a specified value, or regular expression, and returns the position of the match.
2) slice() Extracts a part of a string and returns a new string
3) split() Splits a string into an array of substrings.
4) startsWith() Checks whether a string begins with specified characters.
5) substr() Extracts the characters from a string, beginning at a specified start position, and through the specified number of character.
6) substring() Extracts the characters from a string, between two specified indices.
7) toLowerCase() Converts a string to lowercase letters.
8) toString() Returns the value of a String object.
9) toUpperCase() Converts a string to uppercase letters.
10) trim() Removes whitespace from both ends of a string.

Methods of Array:
1) shift() method removes the first array element and "shifts" all other elements to a lower index.
2) length() property provides an easy way to append a new element to an array.
3) delete() elements can be deleted.
4) splice() method can be used to add new items to an array.
5) slice() method slices out a piece of an array into a new array.
